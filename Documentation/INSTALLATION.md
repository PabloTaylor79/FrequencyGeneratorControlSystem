# Installation Guide for Frequency Generator Control System

## Firmware Installation
1. Download the latest firmware from the [releases page](https://github.com/PabloTaylor79/FrequencyGeneratorControlSystem/releases).
2. Connect your device to the computer using a USB cable.
3. Open the firmware flashing tool and select the downloaded firmware file.
4. Follow the on-screen instructions to complete the installation.

## Desktop Application Installation
1. Download the desktop application installer from the [releases page](https://github.com/PabloTaylor79/FrequencyGeneratorControlSystem/releases).
2. Run the installer and follow the installation wizard to install the application on your computer.
3. Once installation is complete, launch the application to ensure it runs correctly.

## Device Connection Configuration
1. Connect your Frequency Generator to the computer via the USB cable.
2. Open the desktop application and go to the settings menu.
3. Select the appropriate COM port your device is connected to.
4. Save the settings and restart the application if necessary.

## Verification Steps
1. Open the desktop application after configuration.
2. Check the status indicator to ensure the device is connected.
3. Send a test command from the application to the Frequency Generator.
4. Verify that the device responds as expected.

## Troubleshooting
- If the device does not connect, ensure the correct COM port is selected.
- Restart the application and re-verify the USB connection.
- Consult the user manual for further troubleshooting steps.

## Post-Installation Steps
1. Ensure all drivers are up to date.
2. Regularly check for application and firmware updates via the releases page.
3. Review the documentation for any additional configuration settings.
