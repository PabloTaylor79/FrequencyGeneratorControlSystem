# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-22

### Added
- ✅ Complete firmware for STM32H743 microcontroller
- ✅ RF frequency control (10 MHz - 6 GHz)
- ✅ Power control (-20 to +15 dBm)
- ✅ MAX2871 RF synthesizer driver
- ✅ USB CDC virtual COM port communication
- ✅ Real-time monitoring (Temperature, Voltage, Current)
- ✅ Calibration system with FRAM storage
- ✅ FreeRTOS integration
- ✅ Complete WPF desktop application
- ✅ MVVM architecture with Prism framework
- ✅ Program editor with visual timeline
- ✅ Material Design UI theme
- ✅ Comprehensive documentation
- ✅ API reference documentation
- ✅ User manual
- ✅ Troubleshooting guide
- ✅ Architecture documentation
- ✅ CMake and Makefile build system
- ✅ MIT License

### Features
- Frequency range: 10 MHz - 6 GHz
- Power range: -20 to +15 dBm
- Real-time system monitoring
- Program management system
- Automatic calibration
- USB communication protocol
- Temperature/voltage/current monitoring
- Status logging
- Settings management

### Documentation
- Complete installation guide
- User manual with examples
- API reference for all commands
- Troubleshooting guide with 50+ solutions
- System architecture documentation
- Contributing guidelines

### Tech Stack
- **Firmware:** C + FreeRTOS
- **Desktop:** C# + WPF + Prism
- **Communication:** USB CDC/ACM
